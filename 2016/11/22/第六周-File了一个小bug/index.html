第六周-File了一个小bug

发表于 2016-11-22

这一周感觉是过得最快的一周，每天遇到一点小问题，然后稍微解决一下，一天就过去了。每天虽然不知道干了些什么，但是很充实，可能就是瞎忙活吧。

这周工作上，一个小bug解决了几周，本周本来准备提交一下，结果这种出问题。首先是code review没有通过，像这样tricky的解决方案，最好得有注释在这个上面。然后是跑了测试的机器被别人停了，结果还没有拿出来，等于白跑了测试。然后跟amit要来了新机器，又遇到了从11升到12的升级的问题，然后又遇到了一个Flag
Day问题，最后的竟然ZTS没有跑过。就是在这样一堆小问题中一周过去了，bug依旧没有提交。不过这一周学到了不少东西，Amit非常详细的跟我讲解了什么是Flag Day，什么是Consle，什么是ILOM。并且这一周由于各种原因，几个meeting都取消了，不过还是发了一份weekly report给mark。周五下午，NCG举办了游戏活动，猜单词、猜歌曲，这样的游戏对我这样英语不咋的人来说太吃亏了。

生活上，这周吃到了期待已久的螺蛳粉，虽然vela反映没有北京的螺蛳粉先生好吃，但是我感觉已经很不错了，至少够辣，虽然臭味差了一点。这周突然决定了要去阿拉斯加，然后就开始各种扒攻略，看机票和酒店。虽然目前来说可能这趟旅程要被取消了，但是一趟攻略做下来，对阿拉斯加倒是了解不了不少。这周还意外喜欢上了台球，有鹏博带着感觉能学到不少，以前实在是太乱打了，技术太差了！

这周很平淡，没有太多的可说了，希望下一周工作顺利，多干点活！
