新第一周-坚持一件事

发表于 2017-05-07

为什么叫做新第一周呢，因为从这一周开始，我想让自己与过去不太一样，我想开始健身，跑步，早睡，早起，吃好每一顿饭，当然一周里没法全都实现，打算每一周开始坚持一件事，养成一个习惯。有人说21天形成一个习惯，我感觉有时开始做了就自然是一个习惯了。

今天看到巴菲特讲的一句话，很受感触：“Look for the job that you would take if you didn’t need a job.” 他希望年轻人都能去找一份自己热爱的工作，毕竟只有你所热爱的工作，才能让你愿意利用自己闲暇的时间也去做。所以就不得不说到现在的工作了，目前做的事情我可以很明确的说不是我所热爱的，但是毕竟是工作，同时也是一种学习，我只希望能把自己的工作做好，学到东西，然后去迎接下一份我所热爱的工作。

这一周我坚持了一周的健身，每天6点下班去健身房，8点回家吃晚饭，很喜欢这种状态。同时运动也让我更加饥饿和疲惫，回家后能吃的很香，早点睡觉。但是我觉得我对健身方面的知识不是很够，也没有人在旁边指导，所以接下来得自己好好研究一波了。

下一周的目标是一周12点前睡觉，因为自己从上班以来一直晚睡，晚上玩手机，看电脑，早上起不来，一天状态不好浑浑噩噩。我想要改变这种状态，争取做到。

我很喜欢龙洋在超级演说家说的，未来的那个ta一定是在你变得更好的路上遇见的，所以现在要做的，就是努力让自己变的更好。
